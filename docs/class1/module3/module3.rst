Module 3: Expose Service with NGINX Ingress Controller
======================================================

Overview
--------

In this module you will build on the CIS deployment from the previous modules
and expose the "F5-Hello-World" app with NGINX ingress controller.

.. important:: This module and following labs assume Module2/Lab1 were
   atempted and working without issue. This module reuses the deployed objects.
   If you haven't done so be sure to attempt this lab now. You can review the
   content here:

   `Lab 2.1 - Install & Configure CIS in ClusterIP Mode <../module2/lab1.html>`_
